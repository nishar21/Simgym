# AI Shopping Agent

A polished React + TypeScript storefront with local login, product search, cart actions, wishlist support, and an AI-style shopping assistant.

## Run locally

```bash
npm install
npm run dev
```

## Useful commands

```bash
npm run lint
npm run build
```

## Features

- Local sign in / sign up demo with persisted session
- Searchable and filterable product catalog
- Wishlist and add-to-cart actions
- Chat assistant for recommendations, shipping, returns, discounts, payments, order tracking, account help, and cart summaries
- Production build ready for Vite/Netlify
