This project is an Ai Customer Support Agent for Commerce application developed using React. It combines online shopping features with an intelligent chatbot assistant to improve customer experience.

Youtube Link - https://youtu.be/vQQkQMlx2sE

The system allows users to browse products and interact with an AI shopping assistant for instant support related to product recommendations, order tracking, return policies, shipping and cart summary.

The main goal of this project is to make online shopping smarter, faster, and more user-friendly.

Features:
  User Features
  User Login and Logout
  User profile saved using Local Storage
  Product browsing and product listing
  Cart item count display
  AI Chatbot Features
  AI-Based Shopping Assistant
  Product recommendations
  Order tracking support
  Return initiation guidance
  Cart summary display
  Shipping and payment support
  Quick suggestion buttons for common queries
  Widget-based chatbot responses

How It Works:
  User opens the application
  Products are displayed in the storefront
  User logs in and profile is stored locally
  User adds products to cart or wishlist
  Chatbot helps with shopping-related queries
  Chatbot provides recommendations and support
  Products can be added directly from chatbot suggestions

This creates a complete shopping and support experience in one platform.
